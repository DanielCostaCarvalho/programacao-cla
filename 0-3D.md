# Fazer um "Hello world" em uma linguagem pouco usual DESAFIO
## Descrição
Nesse desafio, você precisa exibir em uma tela a mensagem "Hello World".

O desafio consiste em utilizar a "6502 assembly language" para desenhar a mensagem na tela pixel a pixel utilizando o "6502js" que pode ser encontrado nos sites: [MOS 6502 Simulator](https://itema-as.github.io/6502js/), ou [Easy 6502](https://skilldrick.github.io/easy6502/). Ambos os sites possuem documentação ensinando a programar na linguagem.

## Requisitos funcionais
- O programa precisa exibir a mensagem "Hello World" na tela.

## Requisitos não funcionais
- É necessário apontar qual o assembler e simulador foi utilizado (qual site).
- É necessário explicar cada uma das instruções utilizadas.
