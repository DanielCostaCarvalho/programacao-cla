# Criação de um site simples DESAFIO
## Descrição
Nesse desafio, você precisa criar um site simples usando o built-in web server do PHP.

Não é necessário se preocupar com o layout, CSS, ou em como será exibido o formulário, é necessário apenar que seja exibido e que possa ser usado.

## Requisitos funcionais
- O site precisa de uma rota "/" que deve conter um formulário com um input de sugestão e um botão para enviar.
- Também é necessária uma rota "/sugestoes" que recebe um POST com a sugestão preenchida no formulário. Essa rota deve fazer um log da sugestão de alguma forma e redirecionar para a rota "/".

## Requisitos não funcionais
- É necessário um arquivo de documentação ensinando como rodar o projeto.
- É necessária uma documentação de todas as requisições feitas pelo navegador no processo de acessar a página inicial e enviar a sugestão. Também é preciso documentar cada mudança de página.
- É necessário responder a seguinte questão: como é feito o redirecionamento e como o navegador sabe que precisa redirecionar para uma outra página?
