# Criação de um site simples com autenticação DESAFIO
## Descrição
Nesse desafio, você precisa criar um site simples usando algumas das tecnologias já usadas.

Não é necessário se preocupar com o layout, CSS, ou em como será exibido o formulário, é necessário apenar que seja exibido e que possa ser usado.

## Requisitos funcionais
- O site precisa de uma rota "/" que deve conter um formulário com um input de uma chave de API e um botão para enviar um GET para a rota "/restrita-query" com a chave do input como query param.
- Também é necessária uma rota "/401" que deve exibir um  aviso de erro de autenticação.
- Também é necessária uma rota "/restrita-query" que recebe uma requisição GET e deve verificar se a chave recebida coincide com a do servidor e salvar a chave como cookie e deve exibir um link para a página "/restrita-cookie" em caso de sucesso. Em caso de erro, deve ser redirecionada para a página "/401".
- Também é necessária uma rota "/restrita-cookie" que recebe uma requisição GET e deve verificar se a chave recebida via cookie coincide com a do servidor e exibir uma mensagem de sucesso. Em caso de erro, deve ser redirecionada para a página "/401".

## Requisitos não funcionais
- É necessário um arquivo de documentação ensinando como rodar o projeto.
- É necessária uma documentação de todas as requisições feitas pelo navegador no processo de acessar a página inicial passar pela página de query e depois pela de cookie. Também é preciso documentar cada mudança de página.
