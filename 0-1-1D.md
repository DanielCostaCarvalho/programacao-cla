# Criação de um site simples usando AJAX DESAFIO
## Descrição
Nesse desafio, você precisa criar um site simples usando o built-in web server do PHP, mas, dessa vez, usando AJAX e fazendo as requisições e mudanças usando o JavaScript.

Não é necessário se preocupar com o layout, CSS, ou em como será exibido o formulário, é necessário apenar que seja exibido e que possa ser usado.

## Requisitos funcionais
- O site precisa de uma rota "/" que deve conter uma lista de sugestões e um formulário com um input de sugestão e um botão para enviar.

- Também é necessária uma rota "/sugestoes" que recebe um POST com a sugestão preenchida no formulário. Essa rota deve retornar a resposta com o status 201 e um objeto JSON no corpo com um campo de mensagem que deve retornar uma mensagem de sucesso e um campo de sugestão que será outro objeto que deve conter o campo texto com a sugestão enviada. Exemplo:
    "{
        "mensagem": "teste",
        "sugestao": {
            "texto": "teste"
        }
    }"

- O site não pode fazer nenhum redirecionamento, mudança de página, ou atualização. Ele deve apenas fazer a requisição via AJAX, exibir a mensagem na tela brevemente com base na resposta JSON e adicionar a sugestão na lista também com base na resposta JSON.

- As sugestões não precisão ser guardadas em nenhum lugar e nem retomadas. A cada acesso à página principal, é gerada uma nova lista vazia a ser preenchida.

## Requisitos não funcionais
- É necessário um arquivo de documentação ensinando como rodar o projeto.
- É necessária uma documentação de todas as requisições feitas pelo navegador no processo de acessar a página inicial, enviar a sugestão e exibir o novo item na lista.
